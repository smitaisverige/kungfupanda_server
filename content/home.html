This website is about the Kung Fu Panda movie series.(One of my favourite movie series!)

Use the navigation to explore movies and characters and the language.

How to use this website

This website is a Node.js server demonstrating multiple routes, URL queries, external files, and data-driven content.

Available Routes

  • Home: http://localhost:5055/
  • Kung Fu Panda Main: /kfp
  • Movies List: /kfp/movies
  • Characters: /kfp/characters
  • About: /kfp/about

Using Queries

You can pass two queries in the URL to display specific movie information and extra content:

  1. HTML rendered directly in JavaScript:
    The routes generate HTML directly using res.write(). Examples include:
    • /kfp - main Kung Fu Panda page
    • /kfp/about - about page

  2. Content loaded from external files:
    External .txt and .html files are loaded using fs.readFile(). Examples include:
    • /kfp/characters - loads content from characters.txt
    • / - loads content from home.html

  3. URL query parameters:
    The application reads query parameters from the URL to display different content. There are two queries:
    • movie - selects a specific movie
    • lang - language code; default is English (eng)

    Extra info ("English subtitles available") is only shown when lang is NOT English.


    Example clickable links:
    • View Kung Fu Panda 1 (default English) single querry
    • View Kung Fu Panda 2 (English)
    • View Kung Fu Panda 3 (French)
    • View Kung Fu Panda 4 (Spanish)

The Kung Fu Panda route uses data loaded from an external JavaScript file and displays object information dynamically based on the queries.

To test the extra info feature, click a movie link and manually change the lang query in the URL to any value other than eng. You will see the extra info appear.
